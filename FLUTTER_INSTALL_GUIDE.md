# Flutter SDK 安装指导（Windows）\n\n## 步骤1: 下载\n- 访问 [Flutter 官网](https://flutter.dev/docs/get-started/install/windows)\n- 下载 Windows 版 Flutter SDK zip 文件。\n\n## 步骤2: 解压\n- 解压到 `C:\\flutter` (推荐，无空格路径)。\n\n## 步骤3: 配置 PATH\n1. 搜索“环境变量”打开系统属性。\n2. 系统变量 > Path > 编辑 > 新建 > 添加 `C:\\flutter\\bin`。\n3. 保存并**重启终端/VSCode**。\n\n## 步骤4: 验证\n```\npowershell flutter --version\nflutter doctor\n```\n\n## 步骤5: 补充工具\n- 安装 Android Studio (Android SDK)。\n- VSCode 安装 Flutter/Dart 扩展。\n\n安装后运行 `flutter doctor` 分享输出，我继续创建项目。\n\n**注意**: `flutter doctor` 可能提示缺少工具，按需安装。
